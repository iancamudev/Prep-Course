* `for`
* `&&`, `||`, `!`

El bucle for sirve para iterar las veces que queramos. 
Su estructura es la siguiente for (inicio, fin, incremenento) { codigo a ejecutar }

El operador logico && sirve para comparar, para que sea true ambas condiciones tiene que ser verdaderas
El opeador logico || solo es true si alguna de las dos condiciones es verdaderas
El not ! sirve para invertir un booleano, si un valor es !false === true  y  !true === false
